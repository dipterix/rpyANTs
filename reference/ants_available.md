# Check if 'ANTs' is available

Check if 'ANTs' is available

## Usage

``` r
ants_available(module = c("ants", "antspynet"))
```

## Arguments

- module:

  either `'ants'` or `'antspynet'`; default is `'ants'`

## Value

Logical, whether `'ANTs'` or `'ANTsPyNet'` is available
