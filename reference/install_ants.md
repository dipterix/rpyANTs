# Install 'ANTs' via 'ANTsPy'

Install 'ANTs' via 'ANTsPy'

## Usage

``` r
install_ants(python_ver = "3.11", verbose = TRUE)
```

## Arguments

- python_ver:

  'Python' version, see
  [`configure_conda`](http://dipterix.org/rpymat/reference/conda-env.md);
  default is `"3.11"` since `'ANTsPy'` is compiled for all platforms
  under this version

- verbose:

  whether to print the installation messages

## Value

This function returns nothing.
